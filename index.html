Basic HTML Text Tags

Marking up Text in Blocks

When you have text that is displayed in a block, it’s considered a paragraph. Paragraph text can include words or phrases that are emphasized or are of high importance.

We can also reference element names using inline code such as <p>, <h1>, and <section> to talk about their tags directly.

Semantic tags help browsers and assistive technology understand meaning, not just appearance.

Marking Up Text to Convey Meaning

Use semantic elements to give structure and meaning to your content, grouping related ideas into logical sections.

The names of HTML elements are written in angle brackets like <header> and <footer>. Opening and closing tags wrap content to create elements the browser can interpret.

You can draw attention to text with bold or highlight specific terms with highlight. For accessibility, prefer semantic emphasis: use emphasis for stress and importance for strong meaning; the words emphasis and importance are shown here for clarity.
